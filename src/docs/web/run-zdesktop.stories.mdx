# Desktop Apps with Electron
 
<div className="subheading">On root folder in your application</div>

take sure your web application is running on development mode

Run with `yarn`
```sh
    yarn desktop
```

Or run with `npm`
```sh
    npm run desktop
```

### an desktop application will be open embeding your web development mode

- to change address of page renderized, access `elecron/main.js`.
