# Deploy your project to CI
 
<div className="subheading">On root folder in your application</div>

Run with `yarn`
```sh
    yarn build
```

Or run with `npm`
```sh
    npm run build
```

### then 
- your `static html` will be generated on `dist` folder
- commit your code on `develop` branch
- merge your code on `release` branch
- watch your publication on `team channel` on `Slack`
