# Create you documentation with Storybook
 
<div className="subheading">On root folder in your application</div>

Run with `yarn`
```sh
    yarn storybook
```

Or run with `npm`
```sh
    npm run storybook
```

### an browser instance will be open at
```sh
    localhost:6006
```
